# Кейс-задача №4 — Web-версия книжного магазина

Django-приложение с каталогом книг, покупкой и арендой, двумя интерфейсами (пользователь и администратор) и автоматическими напоминаниями об окончании аренды.

## Ссылка на репозиторий GitHub

> **https://github.com/YOUR_USERNAME/bookstore-task4** — замените на актуальную ссылку после публикации.

## Требования

- Python 3.10+
- pip

## Установка и запуск

```bash
cd task4
pip install -r requirements.txt
python manage.py migrate
python manage.py seed_books
python manage.py runserver
```

Откройте в браузере: http://127.0.0.1:8000/

## Функциональность

### Пользователь
- Просмотр каталога и детальной страницы книги
- Сортировка по категории, автору, году (возрастание/убывание)
- Фильтрация по категории
- Покупка книги
- Аренда на 2 недели / 1 месяц / 3 месяца
- Просмотр своих заказов и in-app уведомлений

### Администратор
- CRUD книг (добавление, редактирование, удаление)
- Быстрое редактирование цен, статусов и видимости в каталоге
- Журнал автоматических напоминаний об аренде
- Просмотр просроченных аренд

### Автоматические напоминания
APScheduler запускается при старте сервера (`runserver`) и каждый час проверяет аренды, срок которых истекает в течение суток. Пользователю отправляется email (в консоль при разработке) и in-app уведомление.
